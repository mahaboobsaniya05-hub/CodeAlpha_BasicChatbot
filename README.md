# 🤖 Basic Chatbot using Python

A simple rule-based chatbot developed in Python. It responds to predefined user inputs using conditional statements.

## Features

- Greets the user
- Responds to common questions
- Runs continuously until the user types "bye"
- Handles unknown inputs gracefully
- Beginner-friendly Python project

## Technologies Used

- Python 3

## Concepts Used

- Functions
- While Loop
- If-Elif-Else
- User Input
- String Manipulation

## How to Run

```bash
python chatbot.py
```

## Example Conversation

You: hello
Bot: Hi! How can I help you?

You: how are you
Bot: I'm fine, thanks!

You: bye
Bot: Goodbye!
